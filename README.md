# InsurePro Distributed System

Enterprise-grade distributed insurance quotation system built with:

- Golang (Microservices)
- PostgreSQL (Database per service)
- NATS (Event-Driven Architecture)
- Docker
- React + TypeScript (Frontend)

## Architecture

- Auth Service
- User Service
- Quotation Service
- Notification Service
- API Gateway
- Event Bus (NATS)

## Tech Highlights

- Database isolation per service
- Event-driven communication
- JWT authentication
- Clean architecture
- Dockerized infrastructure
